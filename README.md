# 🩺 Talking Chatbot for Healthcare using Deep Learning

A deep learning-powered healthcare assistant that predicts possible diseases based on symptoms described in plain English. Supports both **text and voice input**, built with Flask, TensorFlow, and NLTK.

---

## 🚀 Features

- 🎤 **Voice Input** - Speak your symptoms using speech recognition
- 🧠 **Deep Learning Engine** - TensorFlow/Keras neural network for disease prediction
- 💬 **NLP Processing** - NLTK tokenization and lemmatization
- 🌐 **Web Interface** - Interactive chat UI built with Flask
- 📋 **Medical Guidance** - Provides first-aid measures and doctor recommendations

---

## 🛠️ Technology Stack

| Layer | Technology |
|-------|-----------|
| Backend | Python, Flask |
| Machine Learning | TensorFlow, Keras |
| NLP | NLTK (Lemmatizer, Tokenizer) |
| Voice | SpeechRecognition, pyttsx3 |
| Frontend | HTML5, CSS3, JavaScript |

---

## ⚙️ Installation & Setup

1. **Clone the repository:**
```bash
git clone https://github.com/CherakaVaishnavi/Talking-Chatbot-for-Healthcare.git
cd Talking-Chatbot-for-Healthcare
```

2. **Install dependencies:**
```bash
pip install -r requirements.txt
```

3. **Run the application:**
```bash
python app.py
```

4. Open browser at: `http://localhost:5000`
